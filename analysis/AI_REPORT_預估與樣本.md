# AI 日報 — 成本估算與輸出樣本

## 💰 成本估算

### 一次日報用多少 token
| 項目 | Token 量 | 說明 |
|---|---|---|
| Prompt(輸入) | ~600 | 系統指示 + 9 顆感測器當日統計 + 任務說明 |
| Response(輸出) | ~400 | 約 250 中文字的日報 |
| **每次合計** | **~1000** | — |

### Claude API 報價(2026 年初常見定價)
| 模型 | 輸入 / 1M tokens | 輸出 / 1M tokens | **每天日報成本** | **每月成本(30 天)** |
|---|---|---|---|---|
| **Haiku 4.5**(推薦) | $1 USD | $5 USD | **$0.0026** | **$0.08** ≈ NT$2.5 |
| Sonnet 4.6 | $3 USD | $15 USD | $0.0078 | $0.23 ≈ NT$7 |
| Opus 4.7 | $15 USD | $75 USD | $0.039 | $1.17 ≈ NT$36 |

**結論:Haiku 4.5 跑日報一個月約 2~3 元台幣,跟一個飯團差不多**。Sonnet 4.6 也只是十元有找。Opus 沒必要,日報 Haiku 完全夠用。

### 比賽期間實際花費
比賽 5 天 + 比賽前測試 5 天 = 10 天 × 1 次/天 = **10 次呼叫**
- Haiku:約 **NT$0.8**
- Sonnet:約 **NT$2.4**

幾乎免費。

### 進階用法的成本
若要改成「每小時呼叫一次」(720 次/月):
- Haiku ≈ NT$60/月
- Sonnet ≈ NT$170/月
仍可接受。

## 📋 輸出樣本(實際代表 Haiku 4.5 會產生的內容)

### Prompt 範例(送進 Claude 的內容)
```
你是水產養殖與藻類培養專家。以下是 2026-04-28 一日的「海水小球藻」(marine Chlorella)監測數據摘要:

資料筆數:5832

各感測器當日統計:
- 溫度(°C): 平均 25.30, 範圍 23.80~26.10, 日內變化 +1.20
- 酸鹼(pH): 平均 8.21, 範圍 7.95~8.45, 日內變化 +0.42
- 溶解(ppm): 平均 982.50, 範圍 950.00~995.00, 日內變化 +18.00
- TDS(EC)(ppm): 平均 27625, 範圍 27170~28080, 日內變化 +320
- 導電(mS/cm): 平均 42.50, 範圍 41.80~43.20, 日內變化 +0.50
- 濁度(NTU): 平均 850.00, 範圍 720.00~1180.00, 日內變化 +460.00
- 光照(lx): 平均 8420, 範圍 0~24500, 日內變化 0
- CO2_B(ppm): 平均 1240, 範圍 850~2100, 日內變化 -180
- CO2_C(ppm): 平均 1180, 範圍 820~2020, 日內變化 -190

請以繁體中文撰寫日報,包含:
1. 總體判斷:藻類狀態(良好 / 普通 / 需警戒),一句話即可
2. 3 個重點觀察:濁度趨勢、pH 變化、CO2 與光照協同等(挑最重要的)
3. 需警戒的異常:若有
4. 明日操作建議:光照、CO2 添加、換水、稀釋等具體動作

控制在 250 字以內,直接寫,不要客套也不要重複數據。
```

### Claude 預期回應(實際 API 還沒呼叫,以下為示意)
```
🌱 藻類狀態:**良好,進入指數生長期前段**

📊 重點觀察
- 濁度全日 +460 NTU(+64%),細胞密度顯著上升,代表養分充足且環境穩定
- pH 早晨 7.95 升至下午 8.45,光合作用消耗 CO2 → 水體鹼化,屬正常規律
- CO2 與光照高度耦合:午後峰值光照時 CO2 降至 820 ppm,光合作用強但 CO2 補給跟不上

⚠️ 需警戒
- CO2 最低 820 ppm 略低,持續可能限制生長速率
- 溫度日變化 1.2°C 可接受,但接近 26°C,夏季要留意散熱

📋 明日建議
1. **白天 12:00~16:00 增加 CO2 供應 30%**,維持 CO2 ≥ 1000 ppm
2. 光照節奏(0~24500 lx)合理,維持
3. 連續觀察濁度,若日增量持續 > 20%,3 天內考慮稀釋以避免進入停滯期
4. 持續記錄 pH 上限,若達 9.0 需警戒

(以上 250 字)
```

## 🔧 怎麼開始用

### 1. 取得 API Key
- 到 [console.anthropic.com](https://console.anthropic.com) 註冊
- 「API Keys」→ 建立新的
- 充值 $5 USD(夠用一兩年)

### 2. 安裝套件
```bash
pip install anthropic
```

### 3. 設定環境變數
**樹梅派 / Linux**:加進 `~/.bashrc`
```bash
export ANTHROPIC_API_KEY="sk-ant-xxxxx"
```

**Windows PowerShell**:
```powershell
$env:ANTHROPIC_API_KEY = "sk-ant-xxxxx"
```

### 4. 測試
```bash
# 不花錢,只印 prompt 看格式
python ai_report.py --preview

# 真正呼叫 API
python ai_report.py
```

### 5. 排程(樹梅派)
```bash
crontab -e
# 每天早上 7:05 跑(在 daily_report 之後)
5 7 * * * cd /home/pi/algae && python3 analysis/ai_report.py
```
